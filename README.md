## 📌 О проекте

**Именинный реестр** — это веб-приложение для учёта людей, которых вы хотите поздравить (белый список) и тех, кого поздравлять не планируете (чёрный список). Все данные сохраняются локально в браузере — никаких серверов, никакой передачи персональных данных третьим лицам.

### ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🤍 **Белый список** | Люди, которых вы планируете поздравить |
| 🖤 **Чёрный список** | Люди, которых вы решили не поздравлять |
| 📊 **Статистика** | Счётчик перемещений в чёрный список (кто чаще всего туда попадает) |
| 👆 **Swipe-жесты** | Свайп влево/вправо по имени — быстрое перемещение между списками |
| 🎂 **Моя дата рождения** | Можно указать и скрыть от посторонних глаз |
| 🌙 **Тёмная тема** | Для комфортной работы в любых условиях |
| ⏰ **Временные метки** | Возможность указать дату/время для каждого человека |
| 📴 **Офлайн-режим** | Работает без интернета (PWA) |
| ⚖️ **Лицензия MIT** | Свободное использование с обязательным указанием авторства |
