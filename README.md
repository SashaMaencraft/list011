## 📌 О проекте

«Именинный реестр» — это веб-приложение для ведения персонального списка людей, которых вы хотите (или не хотите) поздравлять с днём рождения. Приложение сохраняет все данные в вашем браузере (localStorage) и не требует интернета после загрузки.

**🔗 Демо / Исходный код:** [github.com/SashaMaencraft/list011](https://github.com/SashaMaencraft/list011)
Сайт: https://sashamaencraft.github.io/list011/

## ✨ Возможности (функции)

### 📋 Основные списки
| Список | Иконка | Назначение |
|--------|--------|------------|
| **Белый список** | 🤍 | Люди, которых вы поздравляете |
| **Серый список** | 📱 | Люди, которые временно не могут написать / офлайн |
| **Чёрный список** | 🖤 | Люди, которых вы не поздравляете |

### 👥 Управление людьми
- Добавление **друзей** и **родственников** в отдельные блоки
- Быстрый перенос людей между списками (белый ⬄ серый ⬄ чёрный)
- Удаление человека из любого списка
- Редактирование **даты и времени** добавления в список

### 💬 Персональные фразы
- Для **каждого человека** можно назначить **индивидуальную фразу**
- Готовые шаблоны фраз для каждого типа списка:
  - 🤍 **Белый список:** поздравления, пожелания счастья и здоровья
  - 📱 **Серый список:** фразы о временной недоступности («Жду связи», «На связи позже»)
  - 🖤 **Чёрный список:** фразы-пропуски («Игнорируем», «Тишина»)
- Кнопка **«Другое»** — возможность написать **свою собственную фразу**

### 📊 Статистика
- Подсчёт **сколько раз** каждый человек был перенесён в **чёрный** или **серый** список
- Отдельная статистика для каждого типа списка
- Кнопка **очистки статистики**

### 🎂 Настройки даты рождения
- Установка **вашей даты рождения** (формат ДД.ММ.ГГГГ)
- Возможность **скрыть/показать** дату (глазок 👁️/🙈)

### 🎨 Тема оформления
- **Светлая** и **тёмная** темы (переключатель 🌙/☀️)
- Адаптивный дизайн — работает на телефонах и компьютерах

### 💾 Хранение данных
- Все данные сохраняются **локально в браузере** (localStorage)
- Никакие данные не отправляются на сервер
- Кнопка **«Сбросить всё»** — полная очистка всех данных
